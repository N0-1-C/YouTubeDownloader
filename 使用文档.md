# YouTube 视频下载器 - 使用文档

## 快速开始

1. **启动服务** — 双击 `启动下载器.bat`（或开发模式下运行 `python app.py`）
2. **打开页面** — 浏览器自动打开 `http://127.0.0.1:5000`
3. **下载视频** — 粘贴 YouTube 链接 → 点击「解析」→ 选择格式 → 点击「开始下载」

### 设置面板

点击 `⚙️ 下载设置` 展开，包含：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 保存位置 | 视频下载存放目录 | 用户桌面 |
| 手动导入 Cookies | 上传 cookies.txt 文件 | 未配置 |
| 自动打开文件夹 | 下载完成后弹出文件管理器 | 开启 |

## 支持的链接格式

- `https://www.youtube.com/watch?v=xxxxx` — 标准链接
- `https://www.youtube.com/shorts/xxxxx` — Shorts 短视频
- `https://www.youtube.com/embed/xxxxx` — 嵌入式链接
- `https://youtu.be/xxxxx` — 短链接

## 下载格式说明

| 格式选项 | 说明 |
|----------|------|
| 最佳画质 (视频+音频) | 自动选择最高画质，合并为 MP4 |
| 最佳音频 (MP3) | 提取音频，192kbps MP3 |
| 指定分辨率 | 如 1080p、720p、4K 等，含编码信息（H.264/H.265/VP9/AV1） |

## Cookies 导入（下载会员/受限视频时需要）

部分视频（会员专属、年龄限制、私密视频）需要登录状态才能下载：

1. 在 Chrome 安装插件 [Get cookies.txt LOCALLY](https://github.com/kairi003/Get-cookies.txt-LOCALLY)
2. 在浏览器中登录 YouTube
3. 访问 `youtube.com`，点击插件图标导出 cookies 文件
4. 在下载器设置中点击 `📁 选择文件` 导入该文件

> 导入后对所有后续下载生效，无需重复操作。

## 暂停、恢复和取消下载

下载过程中可以随时暂停或取消：

| 操作 | 说明 |
|------|------|
| **⏸ 暂停** | 暂停当前下载，保留进度 |
| **▶ 继续** | 从暂停处恢复下载（yt-dlp 会自动跳过已下载部分） |
| **✕ 取消** | 放弃当前下载，需要重新开始 |

> 暂停后进度会保留。点击「继续」按钮即可从断点恢复。取消后需重新下载。

## 关闭服务

- **网页端** — 点击右上角 `✕ 关闭服务` 按钮
- **命令行** — 直接关闭命令行窗口 / 按 `Ctrl+C`

> 关闭服务后端口 5000 会被释放，可安全重启。

## 常见问题

**Q: 解析失败 / 视频不可用？**
- 确认网络能正常访问 YouTube（需要代理或 VPN）
- 确保链接格式正确
- 受限视频需导入 Cookies

**Q: 下载速度慢？**
- 程序自动检测系统代理，确认代理正常工作
- YouTube 对并发下载有限速，属正常现象

**Q: 音频提取失败？**
- 需要 FFmpeg，打包版已内置。开发模式请确保系统已安装 FFmpeg

**Q: 提示 Node.js 未找到？**
- Node.js 用于解析 YouTube 的签名算法，打包版已内置
- 开发模式需自行安装 Node.js

**Q: 下载到一半报错？**
- 网络不稳定可能导致中断，程序会自动重试（最多 20 次）
- 如果持续失败，尝试更换代理或稍后重试
