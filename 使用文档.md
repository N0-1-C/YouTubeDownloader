# YouTube 视频下载器 - 使用文档

## 快速开始

1. 双击 `启动下载器.bat` 启动服务
2. 浏览器自动打开 `http://127.0.0.1:5000`
3. 粘贴 YouTube 视频链接 → 点击「解析」→ 选择格式 → 点击「开始下载」

## 基本功能

| 功能 | 说明 |
|------|------|
| 视频下载 | 支持选择画质（4K/1080p/720p 等） |
| 音频下载 | 提取 MP3 音频，最高 192kbps |
| 格式选择 | 解析后显示所有可用格式，含文件大小和编码信息 |
| 实时进度 | 下载速度、进度条、预计剩余时间 |
| 下载历史 | 记录所有下载任务，可重新下载已完成的文件 |

## 设置项（点击 ⚙️ 下载设置 展开）

- **保存位置** — 自定义下载目录，默认 `C:\Users\pc\Downloads`
- **手动导入 Cookies** — 用于下载需要登录的会员/私密视频（见下方说明）
- **自动打开文件夹** — 下载完成后自动弹出文件所在目录，默认开启

## Cookies 导入（下载会员/受限视频时需要）

部分视频（如会员专属、年龄限制）需要登录状态才能下载：

1. 在 Chrome 安装插件 [Get cookies.txt LOCALLY](https://github.com/kairi003/Get-cookies.txt-LOCALLY)
2. 在浏览器中登录 YouTube
3. 访问 youtube.com，点击插件导出 cookies 文件
4. 在下载器设置中点击「选择文件」导入该文件

## 常见问题

**Q: 解析失败 / 视频不可用？**
- 确认网络能正常访问 YouTube
- 确保链接格式正确（`youtube.com/watch?v=xxx` 或 `youtu.be/xxx`）
- 受限视频需导入 Cookies

**Q: 下载速度慢？**
- 检查代理设置，本工具自动检测系统代理
- YouTube 对高并发下载有限速，属正常现象

**Q: 音频提取失败？**
- 需要 FFmpeg，已内置。如提示缺失，请确认 `ffmpeg` 文件存在

## 技术信息

- 基于 yt-dlp + Flask，端口 5000
- 自动检测系统代理
- 依赖：Python 3.12、yt-dlp、Node.js、FFmpeg
